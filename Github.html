O QUE É GIT E GITHUB?

GitHub é uma plataforma para gerenciar seu código e criar um ambiente de colaboração entre devs, utilizando o Git como sistema de controle.É lá que você provavelmente vai ter seu repositório e usar no dia a dia.
Já o Git, ferramenta de versionamento local, que salva o estado de cada arquivo na hora do versionamento e caso o arquivo não sofra a alteração, ele cria um link simbólico para o arquivo não editado.

COMO CONFIGURAR O GIT?

Git Tutorial: Configuração e Comandos Básicos

  • Instalar o Git:

    Baixe e instale o Git a partir do site oficial. Siga as instruções de instalação para o seu sistema operacional.

  • Configurar Nome e Email:

    Abra um terminal (ou prompt de comando no Windows) e configure seu nome e email usando os seguintes comandos:

    git config --global user.name "Seu Nome"
                        git config --global user.email "seuemail@example.com"
  • Configurar Editor de Texto (opcional):

    Por padrão, o Git usa o editor de linha de comando para mensagens de commit. Se você preferir usar um editor de texto diferente, configure-o com:

    git config --global core.editor "nome_do_editor"
  • Verificar Configurações:

    Você pode verificar suas configurações usando:

    git config --list
  • Clonar um Repositório Existente:

    Para clonar um repositório existente, navegue até o diretório onde deseja que o repositório seja clonado e execute:

    git clone URL_do_Repositório
  • Criar um Novo Repositório:

    Se você deseja criar um novo repositório local, vá até o diretório do seu projeto e execute: 

    git init
  • Adicionar e Confirmar Mudanças:

    Adicione as mudanças que deseja commitar usando:

    git add arquivo(s)

    E então faça o commit das mudanças:

    git commit -m "Mensagem do Commit"
  • Conectar a um Repositório Remoto:

    Se você deseja conectar seu repositório local a um repositório remoto, use:

    git remote add origin URL_do_Repositório_Remoto
  • Enviar Mudanças para o Repositório Remoto:

    Após adicionar e confirmar suas mudanças, você pode enviá-las para o repositório remoto com: 

    git push origin branch_principal
  • Atualizar seu Repositório Local:

    Para atualizar seu repositório local com as mudanças do repositório remoto, use:

    git pull origin branch_principal

COMO CLONAR UM REPOSITÓRIO?


Criar um Repositório no GitHub:

-Faça login na sua conta do GitHub.
-Clique no sinal de "+" no canto superior direito e selecione "New Repository".
-Preencha o nome do repositório, descrição e outras configurações.
-Clique em "Create Repository".


Clonar um Repositório Existente:

-No GitHub, vá para a página do repositório que deseja clonar.
-Clique no botão "Code" (ou "Clone") e copie a URL do repositório.
-No seu computador, abra o terminal.
-Navegue até o diretório onde deseja clonar o repositório.
-Execute o comando: git clone URL_do_Repositório.
