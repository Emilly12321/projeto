COMO UTILIZAR O GITHUB PAGES?

Crie um Repositório: Comece criando um repositório no GitHub que conterá os arquivos do seu site.(obs:Você pode utilizar um repositório que já possui.Pule esses passos e vá para configurações do repositório).
Você pode nomeá-lo como preferir.
Crie o Site Estático: Coloque os arquivos do seu site estático (HTML, CSS, JavaScript, etc.) no repositório.
Certifique-se de que esses arquivos estejam na raiz do repositório ou em uma pasta específica.

Configurações do Repositório:

Vá para as configurações do repositório.
Role para baixo até a seção "GitHub Pages".
No dropdown "Source" (Fonte), selecione a branch que contém o código do seu site. Por padrão, é a branch main ou master.
Salve as configurações.
Aguarde a Ativação: O GitHub Pages levará um momento para preparar e ativar o site. Quando estiver pronto, você verá um link para o seu site em "GitHub Pages" nas configurações do repositório.
Acesse o Site: Clique no link fornecido para acessar o seu site estático hospedado no GitHub Pages.
